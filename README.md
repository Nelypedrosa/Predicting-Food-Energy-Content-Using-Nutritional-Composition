# Food Composition Data Analysis

## Overview

This project explores the nutritional composition of raw and processed foods using data science techniques. The analysis includes exploratory data analysis (EDA), data preprocessing, regression modeling, and hyperparameter tuning.

## Objectives

- Analyze the nutritional composition of foods.
- Explore relationships among nutritional variables.
- Predict the energy content (kcal) using machine learning models.
- Compare the performance of Linear Regression, Polynomial Regression, and Ridge Regression.
- Optimize the Ridge Regression model using GridSearchCV.

## Dataset

Dataset: Composition of Foods Raw, Processed

Source:
https://www.kaggle.com/datasets/mahdiehhajian/composition-of-foods-raw-processed
